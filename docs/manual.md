# える式台本生成AIエージェント ZERO 活用ガイド

**最終更新日：2026年3月27日**

---

# ZEROとは

ZEROは、YouTube動画の企画立案から台本完成までをAIと一緒に進めるツールです。

える式の台本設計メソッド（視聴者心理に基づいた構成設計）が組み込まれており、あなたの考えや経験をAIとの対話で台本に落とし込めます。

## 台本完成までの流れ

1. **企画を見つける** — フォームに情報を入力 → AIが市場を調査 → 10個の企画案から選ぶ
2. **台本を生成する** — 選んだ企画から台本全文を自動生成
3. **台本を磨く** — 気になる箇所をAIに修正させて完成

# はじめる前に：準備

台本の質を上げるために、2つの準備をしておきましょう。

## 1. プロフィール設定

プロフィール（発信者情報・商品情報・YouTube連携）の設定がまだの方は、[ご利用開始ガイド](getting-started#step-2プロフィールを設定する)を参照してください。

## 2. LINE特典の設計

台本の中には視聴者をLINE登録に導くパートがあります。そのため、**台本を作る前に**LINE特典を設計しておくと、導線まで含めた台本が生成されます。

1. 「**LINE特典作成**」→「**Start**」→「**提案を生成する**」
2. 3つの提案から選択 → 内容を確認・修正 →「**保存してチャットへ**」

# STEP 1：企画を見つける（企画プランナー）

「何について動画を作ればいいか分からない」という状態からスタートできます。フォームに情報を入力すると、AIが市場調査を行い、データに基づいた10個の企画案を提案してくれます。

## 1-1. ヒアリングフォームに入力する

トップ画面から「**企画プランナー**」をクリックすると、ヒアリングフォームが表示されます。チャンネル情報やターゲット、商品情報などを入力してください。

プロフィール設定を済ませていれば「**保存済み情報を読み込む**」で多くの項目が自動で埋まります。

フォームは4つのセクションに分かれています：

| セクション           | 入力する内容                                     |
| -------------------- | ------------------------------------------------ |
| **チャンネル情報**   | チャンネル名、ジャンル、発信内容、目的           |
| **ターゲット**       | 年齢・性別・悩み・理想の状態（最大3ターゲット）  |
| **バックエンド商品** | 商品名、概要、特徴、料金プラン                   |
| **動画設計**         | 一人称、肩書き、実績、撮影スタイル、ベンチマーク |

画面下部に入力進捗バーが表示されます。必須項目をすべて埋めると「**調査を開始する**」が押せるようになります。

## 1-2. AIが市場調査を行う（自動・5〜30分）

「**調査を開始する**」を押すと、AIがあなたのジャンルの市場を自動で調査します。

**この間、ページを離れてもOKです。** 戻ってきたら続きから再開できます。

![20260327_215319](images/20260327_215319.png)

## 1-3. 調査結果を確認する

調査が完了すると、4つのタブで結果を確認できます。

| タブ           | 内容                                                            |
| -------------- | --------------------------------------------------------------- |
| **優先課題**   | 視聴者が抱える悩みの優先ランキング + あなたの強み・NG事項の整理 |
| **インサイト** | 同ジャンルで伸びている動画の共通パターン、季節トレンド          |
| **調査詳細**   | 視聴者の悩み（50件以上）、よくある質問、想定される反論          |
| **参考動画**   | 実際に伸びている動画のタイトル・再生数・チャンネル名            |

確認したら「**企画を作成 →**」をクリックして次に進みます。

![20260327_221239](images/20260327_221239.png)

![20260327_221321](images/20260327_221321.png)

## 1-4. 10個の企画案から選ぶ

調査結果をもとに、AIが10個の企画案をスコア付きで生成します。

各企画カードに含まれる内容：

| 項目                 | 説明                                                                 |
| -------------------- | -------------------------------------------------------------------- |
| **順位・スコア**     | 4軸（視聴者ニーズ・競争優位性・クリック率・再現性）で100点満点で採点 |
| **タイトル案**       | 3パターンのタイトル候補                                              |
| **動画の概要**       | 5〜9項目の構成アウトライン                                           |
| **冒頭15秒の入り方** | 視聴者を掴む最初のセリフ                                             |
| **なぜ伸びるのか**   | 調査データに基づいた根拠                                             |

各企画カードには2つのボタンがあります。

| ボタン | 用途 |
|-------|------|
| **この企画で台本を作成** | 企画の方向性に納得していれば、そのまま台本生成へ |
| **この企画を深掘りする** | 自分の経験や独自の切り口で企画を練り上げてから台本へ |

企画プランナーが提案する企画は市場データに基づいています。そのまま台本にしても十分機能しますが、あなた自身の経験・権威性・エピソードを戦略に組み込むことで、**再生数だけでなく売上にもつながる台本**になります。迷ったら「深掘りする」を選ぶのがおすすめです。

# 企画を深める：企画書作成

企画プランナーで「**この企画を深掘りする**」を選ぶか、トップメニューの「**企画書作成**」から直接アクセスできます。

企画プランナーから進んだ場合は、テーマと切り口が自動で入力されます。

### 企画プランナーとの違い

| | 企画プランナー | 企画書作成 |
|---|---|---|
| **向いている人** | 「何を作ればいいか分からない」 | 「自分なりの切り口がある」 |
| **企画の根拠** | 市場調査データ | あなたの知見・経験 |
| **進め方** | フォーム入力 → 自動生成 | AIと対話しながら深掘り |
| **独自性の反映** | 低め（市場ベース） | 高い（あなたの権威性・エピソードが戦略に組み込まれる） |

### 操作手順

**1.** テーマと、あれば切り口・独自の視点を入力して送信

「こういう角度で攻めたい」「自分のこの経験を活かしたい」という意図があれば、切り口フィールドに入力してください。構成形式（標準 or ランキング）も選べます。

**2.** AIと対話して企画書を完成

AIが対話形式で企画を深掘りします。各フェーズで複数の仮説が提示されるので、ゲージ（1〜5）で評価してください。

| フェーズ | 内容 |
|---------|------|
| **インサイト発見** | ターゲットの「真の悩み」を3つの仮説から特定 |
| **コアアイデア設計** | あなたの権威性・経験を活かした信頼獲得戦略を3パターンから選択 |
| **構成設計** | 標準形式 or ランキング形式で動画の構成を決定 |

完成した企画書から、そのまま台本生成（STEP 2）に進めます。

# STEP 2：台本を生成する

企画プランナーまたは企画書作成から進むと、企画の情報が引き継がれて台本生成が始まります。

## 2-1. アウトラインを確認する

AIが「話の流れ」の設計図（アウトライン）を作成します。修正したい点があればチャットで指示してください。

![20260327_220657](images/20260327_220657.png)

## 2-2. 台本を生成する

「**この内容で台本作成**」をクリックすると、約2〜3分で台本全文が生成されます。

![20260327_220914](images/20260327_220914.png)

# STEP 3：台本を磨く（ドキュメント編集）

生成された台本を推敲して完成させます。

## 推奨：範囲を選んでAIに修正させる

**1.** 修正したい箇所を**ドラッグで選択** → **チャットアイコン**をクリック

![範囲選択](images/edit-select-range.png)

**2.** 「もっと簡潔に」「具体例を追加して」など指示を入力

![指示入力](images/edit-instruction-input.png)

**3.** AIの修正提案をセクションごとに **承認** / **編集** / **却下**

![承認](images/edit-approve.png)

**4.** すべて確認したら「**これで確定**」

![確定](images/edit-confirm-all.png)

## その他の編集方法

- **手動編集**: 直接テキストを修正 → 「**変更を確定**」
- **チャット欄からAI編集**: 範囲を指定せず全体に対して指示。意図しない箇所が変わるリスクがあるため、具体的に指示してください。
- **変更点の表示**: 右上の「**変更点を表示**」で差分確認
- **バージョン管理**: 右上の「**前の版を表示**」→「**このバージョンに復元**」で過去の版に戻せます

> **⚠ 制約**: H1見出し（【タイトル・サムネ】【冒頭】【本編】【エンディング】）は現在編集できません。その下の内容を選択して編集してください。

# その他の機能

## 冒頭台本

冒頭部分だけを、異なる訴求パターンで最大3つ生成します。「冒頭のインパクトを試したい」ときに便利です。

1. 左メニュー「**冒頭台本**」→ 台本または企画案を入力
2. 台本設計を確認 → OKなら「OK」と入力
3. A案・B案・C案から選択

## ショート台本

長尺動画の台本から、ターゲット別のショート動画台本を複数生成します。

1. 「**ショート台本**」→ 長尺台本の全文を貼り付け
2. ペルソナ分析を確認 →「OK」と送信
3. 複数のショート台本案を取得

# よくある質問

<details markdown="1">
<summary>▶ Q. ニューロボルトライティングの構成で生成できますか？</summary>

**A.** 固定構成での自動生成は非対応です。テンプレート通りだと視聴者が既視感を持つため、ZEROはテーマに合わせて柔軟に最適化します。

</details>

<details markdown="1">
<summary>▶ Q. チャット欄から編集指示を出したのに反映されません</summary>

**A.** チャット欄からの指示はAIが編集箇所を判定するため、意図しない箇所が変更される可能性があります。台本を範囲選択してから編集指示を出す方法を推奨します。

</details>

<details markdown="1">
<summary>▶ Q. 台本が1つしか提案されません</summary>

**A.** 戦略検討の結果に応じて最大3案を提示する設計です。必要なときのみ複数案が提案されます。

</details>

<details markdown="1">
<summary>▶ Q. AI編集で意図しない箇所が変更されます</summary>

**A.** 却下または手動修正で対応してください。事前にAIへ変更方針を出力させてから編集指示を出すと、意図しない変更を防ぎやすくなります。

</details>

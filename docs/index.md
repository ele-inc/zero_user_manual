# ZERO ドキュメント

ZEROは、YouTube動画の企画立案から台本完成までをAIと一緒に進めるツールです。
市場調査に基づく企画提案、対話による企画の深掘り、台本の自動生成・AI編集まで、一貫して行えます。

## はじめての方

[ご利用開始ガイド](getting-started)からプロフィール設定・クレジットカード登録を済ませてください。

## 各機能の使い方

[活用ガイド](manual)で企画プランナー、台本生成、ドキュメント編集などの詳しい使い方を確認できます。

## お問い合わせ

Discordの個人チャンネルで `@える`（企画・台本の内容）または `@STAFF SHUN`（操作・契約）までお気軽にどうぞ。
